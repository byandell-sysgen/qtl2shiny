# Shiny QTL2 Shiny App

Shiny QTL2 Shiny App

## Usage

``` r
qtl2shinyApp()

qtl2shinyServer(id, projects_df)

qtl2shinyUI(id)
```

## Arguments

- id:

  shiny identifier

- projects_df:

  static data frame with project information

## Value

No return value; called for side effects.
