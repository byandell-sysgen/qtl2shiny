# Shiny Triad App

Shiny Triad App

## Usage

``` r
triadApp()

triadServer(id, hotspot_list, snp_list, mediate_list, probs_obj)

triadInput(id)

triadOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- hotspot_list, snp_list, mediate_list, probs_obj:

  reactive arguments

## Value

No return value; called for side effects.
