# Read Phenotypes

Read Phenotypes

## Usage

``` r
read_pheno(project_df, class, columns = NULL, peak_df = NULL, legacy = FALSE)
```

## Arguments

- project_df:

  dataframe of project information

- class:

  name of class

- columns:

  name of columns from phenotype file

- legacy:

  use legacy data if \`TRUE\`

## Value

dataframe
