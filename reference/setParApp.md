# Shiny Setup Parameters App

Shiny Setup Parameters App

## Usage

``` r
setParApp()

setParServer(id, project_df)

setParInput(id)

setParUI(id)

setParOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- peak_df, pmap_obj, covar, projects_info:

  reactive arguments
