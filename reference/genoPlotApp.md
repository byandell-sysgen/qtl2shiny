# Shiny Genotype Data App

Shiny Genotype Data App

## Usage

``` r
genoPlotApp()

genoPlotServer(id, hotspot_list, geno_list)

genoPlotInput(id)

genoPlotOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- hotspot_list, pairprobs_obj, patterns, project_df, snp_action:

  reactive arguments

## Value

No return value; called for side effects.
