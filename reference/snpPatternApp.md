# Shiny SNP Pattern App

Shiny SNP Pattern App

## Usage

``` r
snpPatternApp()

snpPatternServer(id, snp_list, allele_info)

snpPatternOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- snp_list, allele_info:

  reactive arguments
