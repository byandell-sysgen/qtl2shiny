# Shiny Hotspot Panel App

Shiny Hotspot Panel App

## Usage

``` r
hotspotApp()

hotspotServer(id, project_df, main_par = NULL)

hotspotInput(id)

hotspotUI(id)

hotspotOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- project_df, main_par:

  reactive arguments

## Value

No return value; called for side effects.
