# Shiny Mediate Plot App

Shiny Mediate Plot App

## Usage

``` r
mediatePlotApp()

mediatePlotServer(id, hotspot_list, mediate_list, probs_obj, project_df)

mediatePlotInput(id)

mediatePlotOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- hotspot_list, probs_obj, patterns, project_df:

  reactive arguments

## Value

No return value; called for side effects.
