# Shiny Scan Panel App

Shiny Scan Panel App

## Usage

``` r
scanApp()

scanServer(id, hotspot_list, snp_list, probs_obj, project_df)

scanInput(id)

scanOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- hotspot_list, project_df:

  reactive arguments

## Value

No return value; called for side effects.
