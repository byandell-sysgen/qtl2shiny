# Shiny Window Parameters App

Shiny module for peak selection.

## Usage

``` r
winParApp()

winParServer(id, hotspot_df, project_df)

winParInput(id)

winParOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- hotspot_df, project_df:

  reactive arguments

## Value

list of inputs and scan summary
