# Shiny Phenotype Read App

Shiny module for peak selection.

## Usage

``` r
phenoReadApp()

phenoReadServer(id, set_par, peak_df, project_df)

phenoReadOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- set_par, peak_df, project_df:

  reactive arguments
