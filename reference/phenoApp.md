# Shiny Hotspot Panel App

Shiny Hotspot Panel App

## Usage

``` r
phenoApp()

phenoServer(id, set_par, win_par, peak_df, pmap_obj, hotspot_df, project_df)

phenoInput(id)

phenoUI(id)

phenoOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- set_par, win_par, peak_df, pmap_obj, hotspot_df, project_df:

  reactive arguments

## Value

No return value; called for side effects.
