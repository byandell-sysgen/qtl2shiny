# Shiny Diplotype Parameter module

Shiny Diplotype Parameter module

## Usage

``` r
dipParApp()

dipParServer(id, hotspot_list)

dipParInput(id)

dipParUI(id)

dipParOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- win_par, phe_mx, cov_df, K_chr, analyses_df, project_df, allele_info:

  reactive arguments

## Value

No return value; called for side effects.
