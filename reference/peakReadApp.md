# Shiny Peak Read App

Shiny module for peak selection.

## Usage

``` r
peakReadApp()

peakReadServer(id, set_par, project_df)

peakReadOutput(id)
```

## Arguments

- id:

  identifier for shiny reactive

- set_par, peak_df, pmap_obj, project_df:

  reactive arguments
